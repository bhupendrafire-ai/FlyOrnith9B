# Example AgentOrinth Task

Goal:

```text
Inspect this repository, check Obsidian first, make one small safe improvement, run focused checks, and summarize the result.
```

Acceptance criteria:

- Obsidian memory is consulted before workspace inspection.
- A compact run note is created under `Agent Runs`.
- The dashboard shows plan, next action, events, logs, and blockers.
- Any risky or global command creates a pending approval instead of running.
